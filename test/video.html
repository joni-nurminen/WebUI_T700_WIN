== Embedding the VLC plugin in your webpage == The VLC plugin can be directly embedded in your webpage : You then have to use the official Mozilla/ActiveX api documented here : http://wiki.videolan.org/Documentation:WebPlugin But, if you need '''controls''' (buttons, slidebar...), you have to use the '''open source revolunet VLCcontrols''' library. * the code resides @github : https://github.com/revolunet/VLCcontrols * you have some demo here : http://revolunet.github.com/VLCcontrols Usage exemple : Note the fullscreen is currently broken in VLC plugin itself (VLC>1.1)
